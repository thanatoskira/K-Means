# K-Means
针对全班37个人(N)，每人4门课程成绩(T)，利用K-Means算法将其分成4类(K)


算法介绍
========

* 从N个文档随机选取K个文档作为质心
* 对剩余的每个文档测量其到每个质心的距离，并把它归到最近的质心的类
* 重新计算已经得到的各个类的质
* 迭代2～3步直至新的质心与原质心相等或小于指定阈值，算法结束

K-Means算法
===========
* 基于划分的分类方法
    * 划分的步骤
        * 通过分割的方式将n个元素或者对象所组成的数据集,通过某种特定方式将其划分成k个部分,每一个部分称为一个簇,并且要求 k <= n
    * 满足条件
        * 每个簇中至少包含一个对象
        * 每个对象有且只能属于一个簇
